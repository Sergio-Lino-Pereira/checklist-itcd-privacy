<h1>Política de Privacidade – Checklist ITCD</h1>

<p>A extensão <strong>Checklist ITCD</strong> não coleta, armazena, transmite ou compartilha dados pessoais. Toda informação processada pela extensão permanece estritamente no navegador do usuário.</p>

<h2>1. Finalidade</h2>
<p>A extensão tem como único propósito ajudar na conferência de dados exibidos nas páginas oficiais do sistema ITCD, gerando um checklist dinâmico e local.</p>

<h2>2. Dados processados localmente</h2>
<p>Todas as informações são lidas diretamente da aba ativa do navegador, sem qualquer envio para servidores externos.</p>

<h2>3. Permissões utilizadas</h2>
<ul>
  <li><strong>activeTab</strong> – Permite acesso temporário ao conteúdo da aba ativa.</li>
  <li><strong>scripting</strong> – Utilizado apenas para executar o checklist dentro da página do ITCD.</li>
  <li><strong>host permissions</strong> – Restringe o funcionamento somente aos domínios do ITCD.</li>
  <li><strong>storage</strong> – Armazena configurações locais do painel de checklist.</li>
</ul>

<h2>4. Ausência de coleta de dados</h2>
<p>A extensão não coleta, vende ou compartilha informações, não usa cookies, não rastreia usuários e não envia dados para a nuvem.</p>

<h2>5. Contato</h2>
<p>Para dúvidas sobre esta política, contate o desenvolvedor responsável.</p>
